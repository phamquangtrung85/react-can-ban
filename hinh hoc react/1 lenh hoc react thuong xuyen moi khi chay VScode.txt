- Cập nhật tham chiếu (Refactor Rename):

+Sau khi đổi tên file, VS Code thường sẽ hỏi bạn có muốn cập nhật các import liên quan hay không. Hãy chọn Yes hoặc Always (tùy thuộc vào cài đặt của bạn).

+Nếu không có hộp thoại nào hiện ra, bạn có thể thực hiện thủ công:

+Mở file component đã được đổi tên.

+Đặt con trỏ vào tên class/function hoặc export chính của component (ví dụ: export default class MyComponent hoặc function MyComponent()).

+Nhấn phím F2.

+Nhập tên mới cho symbol (đảm bảo nó khớp với tên file mới của bạn, ví dụ nếu file là NewComponent.jsx, thì class/function cũng nên là NewComponent).

+Nhấn Enter. VS Code sẽ tự động tìm và cập nhật tất cả các tham chiếu đến symbol này trong toàn bộ dự án của bạn.
